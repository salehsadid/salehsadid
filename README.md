##  About Me

> *"Building at the intersection of silicon and software — where firmware meets frontend."*

I'm a **3rd-year CSE undergraduate at KUET** with a dual engineering identity: I write software that runs on servers **and** design circuits that run on chips. From flashing firmware onto microcontrollers to building full-stack web applications, I pursue both ends of the engineering stack.

Currently deepening expertise in **ASP.NET, Laravel, Flutter**, and **AI-powered embedded applications** — and actively leading hardware projects as **Software Team Lead at Embedded Innovators**.
